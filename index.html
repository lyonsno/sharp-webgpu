This is a work-in-progress port of Apple's SHARP to WebGPU compute shaders. Currently only the ViT-Large backbone (DINOv2, patch_size=16) is implemented. The full pipeline (SPN encoder, monodepth decoder, Gaussian decoder, 3DGS output) is not yet wired. Weights are loaded from a local binary file (~1.3 GB fp16) — you must convert them first (see README).
